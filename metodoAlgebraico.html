Método Algebraico

Función original:
f(a, b, c, d) = ∑m(0, 1, 2, 3, 8, 9, 10, 11)
1. Conversión a términos binarios:
(0000, 0001, 0010, 0011, 1000, 1001, 1010, 1011)
Correspondientes a los minterms: 0, 1, 2, 3, 8, 9, 10, 11
2. Expresión canónica:
f(a, b, c, d) = a'b'c'd' + a'b'c'd + a'b'cd' + a'b'cd + ab'c'd' + ab'c'd + ab'cd' + ab'cd
3. Proceso de simplificación:

= a'b'c'(d' + d) + a'b'c(d' + d) + ab'c'(d' + d) + ab'c(d' + d)
→ Factorización por términos comunes

= a'b'c'(1) + a'b'c(1) + ab'c'(1) + ab'c(1)
→ Aplicando ley: d' + d = 1

= a'b'c' + a'b'c + ab'c' + ab'c
→ Eliminando 1 innecesarios

= a'b'(c' + c) + ab'(c' + c)
→ Nueva factorización

= a'b'(1) + ab'(1)
→ Ley: c' + c = 1

= a'b' + ab'
→ Reducción

= (a' + a)b'
→ Factor común

= (1)b' = b'
→ Resultado final
Función simplificada: f(a, b, c, d) = b'
